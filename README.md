# BerryTools - Коллекция инструментов

Здесь собраны небольшие, но полезные веб-инструменты для работы с изображениями и смежных задач — всё работает прямо в браузере и не требует установки.

## 🛠️ Доступные инструменты

### 1. [Sticker Slicer Pro](https://anonim4ik3327.github.io/berrytools/splitterv2.html)
Профессиональный инструмент для нарезки стикеров и изображений на части. Идеально подходит для создания наборов стикеров, разбивки больших изображений на плитки и подготовки изображений для публикации.

**Возможности:**
- Нарезка изображений по сетке
- Настройка количества строк и столбцов
- Предварительный просмотр результата
- Экспорт отдельных частей

---

### 2. [Background Remover](https://anonim4ik3327.github.io/berrytools/background-remover.html) 🆕
Инструмент для удаления фона с изображений. Поддерживает как хромакей, так и автоматическое обнаружение однотонного фона.

**Возможности:**
- **Режим хромакея:** удаление зелёного или любого другого цветного фона
- **Умный режим:** автоматическое определение и удаление однотонного фона по углам изображения
- **Сила обрезки краёв:** удаляет цветные точки и ореолы по краям объекта (0–20 пикселей)
- **Допуск цвета:** настройка диапазона удаляемых цветов (1–100)
- **Сглаживание краёв:** делает переходы более естественными (0–10 пикселей)
- **Контраст границ:** улучшает чёткость перехода между объектом и фоном (0–100)
- **Высококачественное вырезание:** сохранение деталей и полупрозрачных областей
- **Экспорт в PNG:** скачивание результата с прозрачным фоном

**Как использовать:**
1. Перетащите изображение в область загрузки или кликните для выбора файла
2. Выберите режим работы (хромакей или умный фон)
3. Настройте параметры для лучшего результата
4. Нажмите "Обработать" для применения изменений
5. Скачайте результат в формате PNG

**Советы для лучшего результата:**
- Для зелёного хромакея используйте цвет #00ff00 по умолчанию
- Если виден зелёный ореол вокруг объекта, увеличьте "Силу обрезки краёв"
- Для сложных случаев настройте "Допуск цвета" и "Контраст границ"
- Используйте "Сглаживание краёв" для более естественного вида

### 3. [SegImg — Сегментация изображений](https://anonim4ik3327.github.io/berrytools/segimg.html)
Инструмент для автоматической и ручной сегментации объектов на изображении. Подходит для выделения объектов, создания масок и подготовки изображений для дальнейшей обработки.

**Возможности:**
- Автоматическая сегментация объектов на основе цвета/контуров
- Генерация маски (альфа) и отдельного слоя с прозрачностью
- Ручная корректировка маски с помощью кисти (дозаливка/стирание)
- Настройки порога и чувствительности для точной сегментации
- Удаление мелких артефактов и сглаживание краёв
- Экспорт маски в PNG (с прозрачностью) и экспорт сегментов как отдельные изображения
- Поддержка сохранения результата локально без отправки на сервер

**Как использовать:**
1. Загрузите изображение перетаскиванием или через выбор файла
2. Нажмите "Сегментировать" для автоматического выделения объектов
3. При необходимости используйте кисть для доработки маски (добавить/удалить)
4. Настройте порог/сглаживание для более аккуратного результата
5. Скачайте маску или отдельные сегменты в формате PNG

**Советы:**
- Для сложных фонов увеличьте детализацию/чувствительность сегментации
- Используйте ручную корректировку, чтобы убрать мелкие артефакты
- Экспортируйте маску и применяйте её в графическом редакторе для дополнительных правок
